Sollin Algorithm 使用說明

執行後輸入測資檔名後就會輸出Minimal Spanning Tree (可能是其中一個case 或是 Minimal Spanning Forest)


程式演算法說明

先將資料放進二維陣列
然後以循序的去尋找每個node聯外的最短路徑並且記錄連出去的該點(minimum() 的作用)
並且分組。

假設某個未分組的點經過minimum的計算, 其聯外的最短路徑的目的點也是未分組的。
則將這兩個點分成一組

假設某個未分組的點經過minimum的計算, 其聯外的最短路徑的目的點是已分組的。
則將這個未分組的點分到該組。

如果此點已經分組過了 則跳過minimum的計算

經過以上的操作 會產生一些forest

接下來再以這些group為基礎，依序向外連，去判斷連到其他組的最短路徑並且連起來

最後就會產生MST了
